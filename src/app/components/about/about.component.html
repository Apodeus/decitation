Description du jeu

Chaque jour, une nouvelle citation chiffrée vous est proposée.
Déchiffrez-la le plus vite possible et partager votre score avec vos amis !

Principe du jeu

Le chiffrement réalisé est un alphabet de chiffrement. Chaque lettre de l'alphabet est remplacée par une autre lettre.
Ainsi, tous les A peuvent devenir des X et tous les B des M, par exemple.
